Bank marketing campaign — term deposit EDA
ExcelPower BIStatistics

Exploratory data analysis on a ~45,000-row bank telemarketing dataset from a 2017 term-deposit campaign. Built a multi-sheet Excel workbook for bivariate analysis (CORREL matrices, AVERAGEIF / COUNTIFS), created derived features such as age-group bins, and built Power BI DAX measures including percentage-of-total breakdowns.
